# ZXingCodeDemo

###（Android）\<br> 
一个关于二维码扫描的Demo（Android Studio版本）\<br> 
#####*功能：\<br> 
    1：二维码扫描\<br> 
    2：相册中选取图片\<br> 
    3：开启闪光灯\<br> 
    4：历史记录（只放了个按钮，功能未实现）\<br> \<br> 
  
#####使用方法: \<br> 
    直接拷贝整个modlue：libraryzxing，然后关联到你的项目就可以使用了！ \<br>  \<br>
